# Tic Tac Toe

A small browser-based **Tic Tac Toe** game built with **HTML, CSS, and JavaScript**, focusing on game logic, and DOM manipulation.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Two-player gameplay
- Turn-based logic
- Win and draw detection
